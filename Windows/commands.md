# Common Commands

## User Details

* `whoami`
* `whoami [/user] [/groups] [/claims] [/priv] [/all]`
* `net user [/username]`

## Machine Details
* `hostname`
* `systeminfo [/s <computer>] [/u <domain>\<username> [/p <password>]`
* `tasklist`

## Network Details

* `arp /a`: Arp cache
* `ipconfig`
* `gpresult /u <user> /p <password>`: Get policies for machine/user
